# Phase 4d — Deployment (Day 17)

Deploy to Vercel (or self-hosted). Steps: 1) Production build (`npm run build`), 2) Configure Vercel project with env vars, 3) Set up Supabase production project, 4) Configure custom domain/DNS, 5) Smoke test on production URL, 6) Create super admin account, 7) Seed platform_config with default values. See DEPLOYMENT_GUIDE.md for detailed steps.
