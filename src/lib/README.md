# /src/lib — Utility Functions & Clients

This directory contains utility functions, service clients, and calculation engines.

## Files (created by Teammate 1):
- `supabase.ts` — Supabase browser + server client (Step 5)
- `engines.ts` — Calculation engines: pricing, deposits, fees, payouts (Step 6)

## Rules

- TypeScript strict mode — no `any` types
- Export named functions, not default exports
